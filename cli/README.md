# auto-pr CLI

全自动 PR 提交工作流 CLI 工具。

## 安装

```bash
npm install -g auto-pr-workflow
```

## 使用

```bash
auto-pr init      # 初始化项目配置
auto-pr submit    # 提交 PR
auto-pr watch     # 监控 CI
auto-pr review    # 触发 AI 审查
auto-pr merge     # 启用 auto-merge
auto-pr describe  # 自动生成 PR 描述
```
