## Learning
- **Symlink escape in path resolution**: When walking up to find existing ancestor and rejoining suffix, intermediate paths that exist (e.g. symlinks) must be canonicalized before continuing. Otherwise a symlink in the suffix can escape workspace boundary.
- **Flaky test pattern**: Tests using fixed filenames in shared temp dirs are flaky. Use PID or UUID-based unique dirs.
